# Discipline Score — розбір з моїми показниками

Джерело: https://ftmo.com/en/blog/consistency-is-very-important-for-success-in-trading/

---

## Головна ідея

Довгостроковий успіх у трейдингу = стабільність, а не разові великі виграші.

Будь-яка стратегія проходить через прибуткові і збиткові серії. Прибуток з'являється тоді, коли прибуткові угоди перевищують збиткові — **в кількості або в обсязі**.

---

## Два підходи — порівняння

| | Стратегія A | Стратегія B |
|---|---|---|
| Прибуток | 50% за 5 угод | 30% за 100 угод |
| Макс DD | 50% | 10% |
| Надійність | Низька — ризик знищення | Висока — стабільно |
| Підходить для FTMO | ❌ | ✅ |

**Висновок:** краще заробляти менше, але стабільно. Стратегія з 50% DD — одна погана серія і рахунок знищено.

---

## Формула Discipline Score

```
Discipline Score = (1 – |найкращий або найгірший день| / |сума всіх днів|) × 100%
```

- Береться **абсолютне значення** найбільшого виграшного або збиткового дня
- Ділиться на **суму абсолютних результатів** всіх торгових днів
- Результат: чим рівніший розподіл — тим вищий скор

### Діапазони:
| Скор | Колір | Значення |
|---|---|---|
| 0–30% | 🔴 Червоний | Дуже нерівномірно |
| 30–80% | 🟡 Жовтий | Fair — є над чим працювати |
| 80–100% | 🟢 Зелений | Відмінна стабільність |

### Мій Discipline Score: **40% (Fair 🟡)**

**Розрахунок:**

| День | Угод | Лотів | Результат |
|---|---|---|---|
| 27 квіт | 3 | 0.20 | +$154.85 |
| 28 квіт | 8 | 2.71 | +$228.57 |
| **Сума** | **11** | **2.91** | **+$383.42** |

```
Найкращий день: $228.57
Сума всіх днів: $383.42

Score = (1 – 228.57 / 383.42) × 100% = (1 – 0.596) × 100% = 40.4%
```

**Причина низького скору:** 28 квіт дав 59.6% всього прибутку — 8 угод проти 3 на наступний день. Дуже нерівномірно.

## Як виглядав би ідеальний сценарій

Якби я торгував рівномірно по ~$120–130/день:

| День | Результат |
|---|---|
| День 1 | +$120 |
| День 2 | +$125 |
| День 3 | +$118 |
| День 4 | +$130 |
| День 5 | +$122 |
| **Сума** | **+$615** |

```
Найкращий день: $130
Сума: $615

Score = (1 – 130 / 615) × 100% = (1 – 0.211) × 100% = 78.9% → майже зелений
```

При 3 угодах/день × expectancy $34.86 ≈ $105/день → після 6 днів скор був би ~78%.

## Що знижує Discipline Score

1. **Overtrading** — 8 угод в один день проти 3 в інший
2. **Нерівномірний розмір позицій** — 2.71 лота vs 0.20 лота
3. **Revenge trading** — спроба "відігратись" після збиткового дня великими позиціями
4. **Один великий виграш на тлі малих** — навіть якщо все прибуткове
